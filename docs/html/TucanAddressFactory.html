Documentation

TucanAddressFactory provides a base factory implementation for servants to inherit from. This is done so that addresses can be given out by their base class TucanAddress. This is handy when extending the system.
TucanAddress create() raises(TucanException)
The create() method will return a new reference to a TucanAddress object. This is so that many different addresses can exists at the same time. If something strange goes wrong this method will raise a TucanException.
