Documentation

The TucanTransportFactory interface is a base definition for services to inherit from and implement. It handles the creation of new object references.
TucanTransport create()
The create() method will create a new reference to a transport object and return it. This is useful so that multiple transports can exist at the same time. If an error occurs in this method a TucanException should be raised.
