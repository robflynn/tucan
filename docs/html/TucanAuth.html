Documentation

TucanAuth is the class wrapper for user data that is needed for the authentication process. It has the capabilty of taking a pin number or a password for authentication but both are not needed. So if you only want to use the passward capabilites then do so and do not concern yourself with the pin numbers. A TucanAuth object is generally only gotten from a TucanSession object so make sure that you have become familar with that interface as well.
