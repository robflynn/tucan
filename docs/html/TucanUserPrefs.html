Documentation

TucanAttributeSeq getAllAttributes() raises(TucanException)
getAllAttributes() takes a user name as a string and returns TucanAttributeSeq containing all preferences for the user. A TucanException will be raised on error

string getAttribute(in string key) raises(TucanException)
getAttribute() takes a user name and a key as a string and returns a value as a string. It raises an exception on error.

void setAttribute(in string key, in string value) raises(TucanException)
setAttribute() takes a user name, key, and value as a string. It sets the desired prefernece in the database. If the preference previouisly exists it will be updated with the new information. A TucanException will be raised on error.

void deleteAttribute(in string key) raises(TucanException)
deleteAttribute() takes a name and key as a string. It attempts to delete the preference entry from the database. A TucanException will be raised on error.

void setAuth(in TucanAuth myAuth) raises(TucanException)
setAuth() takes an Auth object and sets it as the current auth object for the UserPrefs object

TucanAuth getAuth() raises(TucanException)
getAuth() returns the TucanUserPrefs object's current Auth Object

void destroy()
destroy() removes the TucanUserPrefs object and frees the memory associated with it. YA client should always call this method in order to prevent memory leaks.
